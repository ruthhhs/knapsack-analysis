## Analisis Perbandingan *Dynamic Programming*, *Greedy*, dan *Genetic Algorithm* untuk Menyelesaikan Permasalahan *Knapsack* pada Pemilihan *Merchandise* Event Anime/Jejepangan 

### Abstarct

Meningkatnnya minat masyarakat terhadap anime dan budaya Jepang mendorong semakin banyaknya event yang menyediakan berbagai macam merchandise bertema anime. Permasalahan pemilihan merchandise untuk memperoleh kepuasan maksimum dengan keterbatasan anggaran dapat dimodelkan sebagai 0/1 *Knapsack Problem*. Penelitian ini membandingkan tiga algoritma penyelesaian, yaitu *Dynamic Programming*, *Greedy Algorithm*, dan *Genetic Algorithm*. Berdasarkan hasil analisis dan pengujian, *Dynamic Programming* menghasilkan solusi yang paling optimal dibandingkan kedua algoritma lainnya.

### Keywords

Dynamic Programming, Greedy, Genetic Algorithm
